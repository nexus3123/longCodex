# 《反转千金：猎物与猎手》故事大纲

**字数预估：** 15000 - 20000字
**题材类型：** 现代都市 / 女性视角 / 爽文反击 / 职场校园
**核心标签：** 真假千金、极限拉扯、智商碾压、职场生存、绝对利己

---

## 核心冲突与爽点设计 (基于 Novelist Skill 规则)

- **核心矛盾：** 假千金（苏婉婉）常年利用“白富美”人设在学校与职场汲取资源并进行职场霸凌；真千金（林秋）深谙底层生存法则，带资回归，但不公开身份，选择在暗处作为“猎手”对苏婉婉进行资源剥夺与社会性绞杀。
- **爽点设计：**
  - **信息差碾压：** 林秋以“贫困实习生”身份入局，苏婉婉以为自己在霸凌底层，实则林秋在收集她的学术造假与职场贪贿证据。
  - **连环局反杀：** 不依赖泼妇骂街和扯头发，林秋在职场利用苏婉婉的贪婪，设下“巨额回扣”陷阱，让其作茧自缚。
  - **情感斩断：** 面对偏心假千金的势利养母与所谓的精英学长，林秋毫无圣母心，该利用时利用，最后全部踢开。
- **背景语境设计 (虚构映射现实)：**
  - **绝不出现真实地名：** 所有地名皆为化名，但能让读者一秒代入（如金融中心“华海市”代指魔都/陆家嘴，“京北大学/京北T大”代指顶级双子星高校，“海淀/中关村”化名为“云创科技园区”等）。
- **基调与防崩：**
  - “盐溶于汤”：林秋的狠辣不靠内心独白，通过她面对刁难时不动声色的反应和致命一击体现。
  - 绝不降智：苏婉婉不是无脑反派，她善于伪装和利用舆论，林秋必须用更严密的逻辑和阳谋将其逼入死角。

---

## 主要角色档案 (3D 立体化构建)

### 林秋 (女主 - 真千金 / 猎手)
- **核心标签：** 极致利己、底层食腐者、智商在线的疯批。
- **性格底色：** 在孤儿院和社会底层摸爬滚打，信奉“所有关系都是资源置换”。可以容忍黑暗和屈辱，只要这能换来翻盘的筹码。
- **反差细节：** 表面上是个总是戴着黑框眼镜、穿着廉价起球毛衣的唯唯诺诺实习生/室友；但她在进行几百万收购案交锋时，会顺走会议桌上的免费薄荷糖。
- **动机：** 拿回属于自己的一切，并让苏婉婉为过去的掠夺付出代价。

### 苏婉婉 (反派 - 假千金 / 猎物)
- **核心标签：** 伪善名媛、精致利己、极度渴望认同。
- **性格底色：** 知道自己是假千金后，极度恐惧失去一切。她的作恶动机基于“生存恐惧”——她必须把真千金踩在脚下，才能证明自己更有价值。
- **反差细节：** 朋友圈全是岁月静好的画展与高定下午茶；私下里会因为极度的嫉妒和不安，亲手撕碎室友的高档面试装。
- **动机：** 守住豪门地位和未婚夫资源，利用职场和学术圈打造不可替代的社会壁垒。

### 顾青寒 (男主 - 大学导师 / 顶级风投合伙人)
- **核心标签：** 斯文败类、极度理智的资本捕食者。
- **性格底色：** 眼底只有价值的冷血精英。他不是传统意义上被女主单纯吸引的“霸总”，而是林秋的“高阶同类”。前期他因为苏婉婉背后的豪门资源而做了她的学术靠山；后期在审视中发现了林秋惊人的商业嗅觉和手腕，果断倒戈。
- **与女主的情感拉扯：** 林秋和顾青寒之间是“双强博弈”的非传统恋爱。两人在职场上极限试探，互相防备又彼此欣赏，最终结成绝对利益至上的狩猎同盟。他的爱，建立在智商的势均力敌之上。

### 沈辞 (男二 - 同系高岭之花 / 默默守护者)
- **核心标签：** 理想主义、温润寡言、暗中破局者。
- **性格底色：** 天才计算机系学神，出身干净优渥，是世俗眼中的天之骄子。他看透了苏婉婉的装腔作势，却因为偶然窥见了林秋在天台独自崩溃又迅速冷静的脆弱瞬间，而产生了极强的保护欲。
- **不可替代的作用：** 他绝不是无脑的舔狗，而是女主复仇计划中最硬核的技术后盾。当林秋在职场或学术上面临监控被删、数据被篡改的绝境时，沈辞总能在不惊动任何人的情况下，通过追踪IP或恢复底层数据，递给林秋最致命的翻盘底牌。他对林秋的爱是无声而克制的。

### 寝室三人组 (配角B面)
- **赵晓萌：** 墙头草女孩，前期为了奖学金名额讨好苏婉婉，后期被苏婉婉当成学术造假的替罪羊时，拼死反咬一口（拒绝工具人化，有强烈的自救本能）。
- **徐佳：** 独善其身型，看破不说破，在关键时刻为了交换顶尖风投公司的内推名额，向林秋提供了苏婉婉篡改答案的致命录音（基于利益交换的同盟）。

## 剧情大纲规划 (黄金三章与节奏控制)

### 第一阶段：猎物入局 (约 3000 字)
- **起稿冲突 (黄金前三章)：** 
  - **场景：** 顶级风投公司（专注企业投资与包装）的实习生转正答辩会前夕。
  - **情节：** 苏婉婉在寝室和公司双重施压，暗中利用特权删除了林秋的答辩原始数据。林秋并未惊慌哭闹，而是通过暗中关注她的男二**沈辞**，精准恢复了底层数据日志，甚至拿到了苏婉婉篡改记录的IP证据。
  - **爽点反杀：** 答辩现场，林秋用一份“更具颠覆性”的底层市场调研方案反杀，并巧妙将数据异常归结为“测试漏洞”。这种狠辣和应对自如，瞬间吸引了原本是苏婉婉靠山的男主**顾青寒**的注意力，顾青寒的眼神从漫不经心变为锋利的审视。

### 第二阶段：寝室与校园的微观绞杀 (约 5000 字)
- **信息差游戏：** 林秋回到寝室，继续扮演被欺压的贫困生。苏婉婉为了切断林秋的后路，在顾青寒面前造谣林秋“学术作风有问题”。
- **同盟与反击：** 
  - 林秋不解释，直接以“帮忙做数据”为由，介入苏婉婉负责的核心课题。
  - 沈辞作为实验室的学神，表面不问世事，实则在代码层面为林秋开了后门，让林秋完全掌握了苏婉婉挪用课题经费和数据造假的致命死局。
  - **爽点爆发：** 在全校公开路演时，学术委员会当场拆穿苏婉婉的谎言。导师顾青寒为了保全风投公司的声誉和自身利益，冷漠地当众切断了与苏婉婉的学术绑定，并顺势抛出橄榄枝，要求林秋成为他的直属下属（两人正式形成极限施压的上下级关系）。

### 第三阶段：职场资本局的阳谋 (约 7000 字)
- **战场转移：** 冲突升级至家族企业与风投公司的深度合作案。苏婉婉试图挪用公司资金填补自己造假的窟窿，并联合未婚夫学长设局。
- **双雄布局（男女主拉扯）：** 
  - 林秋正式拿到豪门资源，但她并未直接打脸，而是作为“第三方资方代表”身份重新坐在顾青寒和苏婉婉对面。
  - 林秋设下一个“巨额回扣”诱饵，苏婉婉的贪婪被无限放大。在这个过程中，顾青寒和林秋没有一句多余的废话，两人仅凭商业直觉就在谈判桌上达成了绞杀苏婉婉的默契配合。
  - **男二危机救援：** 苏婉婉狗急跳墙雇人抹黑林秋的金融信誉，沈辞在关键一晚黑掉了公关公司的全部通稿后台，将苏婉婉的录音置顶全网。

### 第四阶段：收网与深渊 (约 3000 字)
- **认亲宴 / 宣判现场：** 
  - 家族认亲宴上，苏婉婉准备利用自己的商业成果做最后挣扎。
  - 林秋拿着司法审计报告和职场受贿铁证入场。没有撕扯，只有最平静的语气，宣判苏婉婉的社会性死亡与牢狱之灾。
  - **大结局拉扯：** 苏婉婉失去一切被带走。林秋走出宴会厅，沈辞安静地在雨中为她撑伞，递给她最终的底牌备份，微微一笑深藏功与名。而长街另一头，黑色迈巴赫车窗降下，顾青寒举起酒杯，对这位新晋的顶级掠食者遥遥致意。
  - 林秋没有走向任何一个人，她独自踏入雨幕，迎向属于自己的一切。全篇在绝对冷静和掌控权力的双强张力中收尾。
